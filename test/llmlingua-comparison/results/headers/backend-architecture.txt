### Dependency Flow
```text
Infrastructure -> Application -> Domain
      |               |            |
      v               v            v
 (External)     (Use Cases)   (Business Rules)
```text

**Key Principles:**

- Dependencies flow inward (toward Domain)
- Domain layer has no external dependencies
- Each layer only knows about layers it depends on
